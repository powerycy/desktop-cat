# 桌面健康守护宠物 — 产品方案

## 产品背景

长时间伏案工作导致久坐不动，是程序员和白领群体的普遍健康隐患。研究表明每 40-50 分钟起身活动 5 分钟，可有效缓解腰椎压力、改善血液循环、降低 VTE 风险。但工作沉浸时往往忽略时间流逝，传统定时器提醒生硬且易被忽略。

## 核心需求

> 工作过于投入，不利于身体健康。需要一个**不突兀、有情感连接**的方式，让用户每工作 40 分钟主动休息 5 分钟、起来活动。

## 产品定位

一只常驻桌面的像素宠物，平时安静陪伴工作，到时间时用"宠物需要你休息"的情感化方式温柔打断，引导用户离开屏幕活动身体。

## 产品架构

```
┌──────────────────────────────────────────────┐
│              桌面透明窗口（宠物层）             │
│   像素动画 · 状态机驱动 · 自由行走/坐卧/睡觉   │
└──────────────────────┬───────────────────────┘
                       │
┌──────────────────────▼───────────────────────┐
│              休息提醒引擎（计时 + 全屏遮罩）     │

│              Tauri 原生层（Rust）              │
│   系统托盘 · 多窗口管理 · 开机自启 · 配置持久化  │
└──────────────────────────────────────────────┘
```

## 核心功能

### 1. 桌面陪伴宠物
- 透明无边框窗口，宠物常驻桌面最上层
- 状态机驱动的自然行为循环：坐着待机(50%) → 睡觉(40%) → 行走(10%)
- 像素风逐帧动画，支持翻转朝向
- 鼠标拖拽移动 + 点击互动反馈
- 点击穿透：非宠物像素区域不阻挡正常操作

### 2. 智能休息提醒
- 可配置工作时长（默认 40 分钟）和休息时长（默认 5 分钟）
- 休息触发时宠物从屏幕边缘潜行入场，全屏显示休息动画
- 倒计时结束自动恢复工作状态
- 紧急情况支持 Alt+F4 强制退出

### 3. 休息模式全屏动画
- 三阶段演出：潜行入场 → 趴下过渡 → 趴平呼吸
- 独立于日常动画的 rest 专用帧集
- 引导用户真正离开屏幕（全屏遮罩无法操作其他应用）

### 4. 自定义形象系统
- 内置默认像素猫形象
- 支持上传自定义 PNG 帧集替换宠物外观
- 基于 manifest.json 的动画配置，帧数自动识别
- 设置界面预览 + 一键切换形象
- 大小缩放可调（0.5x ~ 3x）

### 5. 系统集成
- 开机自动启动（可选）
- 系统托盘常驻，右键菜单快捷操作
- 极低资源占用（Tauri 原生框架，内存 < 30MB）

## 技术栈

| 层 | 技术 | 选型理由 |
|----|------|---------|
| 前端 | Vue 3 + TypeScript | 组件化 + 响应式状态管理 |
| 桌面框架 | Tauri v2 (Rust) | 极致轻量，内存占用仅 Electron 的 1/10 |
| 动画系统 | 逐帧 PNG + Canvas 渲染 | 像素风完美呈现，支持点击穿透 |
| 状态管理 | Pinia | 宠物行为状态机 + 配置持久化 |
| 窗口管理 | Tauri 多窗口 API | 透明窗口 + 全屏休息窗口 + 设置窗口 |

## 关键设计决策

1. **选择 Tauri 而非 Electron** — 安装包 < 5MB，内存 < 30MB，适合长期后台常驻
2. **状态机驱动行为** — 自然的行为过渡（走 → 坐 → 睡），宠物"活着"的感觉
3. **全屏遮罩式休息** — 不是弹窗提醒而是强制休息，确保用户真正离开屏幕
4. **点击穿透技术** — Canvas 逐像素检测透明区域，宠物窗口不影响正常工作

## 用户体验流程

```
启动应用
  │
  ▼
宠物出现在桌面角落，安静陪伴
  │ （行走/坐着/打瞌睡 自然循环）
  │
  ├── 用户点击宠物 → 互动动画反馈
  ├── 用户拖拽宠物 → 移动到新位置
  │
  ▼  [40 分钟工作计时结束]
  │
全屏：宠物从右侧潜行入场 → 趴下 → 呼吸循环
  │  屏幕显示倒计时 5:00
  │  用户起身活动……
  │
  ▼  [5 分钟休息结束]
  │
自动恢复，宠物回到桌面继续陪伴
  │
  ▼  [循环]
```

## 产品价值

- **健康守护**：强制休息机制，有效预防久坐损伤
- **情感化设计**：不是冰冷的闹钟，而是"宠物需要你陪它休息"
- **零干扰陪伴**：工作时安静不打扰，点击穿透不影响操作
- **极致轻量**：Tauri 架构确保长期后台运行无感知
- **个性化**：支持自定义形象，满足不同审美偏好
